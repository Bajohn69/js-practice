## 安裝

Node.js 版本建議為：`16.18.0` 以上

### 安裝套件

```bash
npm install
```

### 環境變數設定

新增 .env 檔案，內容如下

```bash
REACT_APP_ID=64ef76a6
REACT_APP_KEY=23f652e912d849de03f109d299c0fb91
```

### 運行專案

```bash
npm start
```

## 專案技術

- axios v1.2.1
- react v18.2.0
